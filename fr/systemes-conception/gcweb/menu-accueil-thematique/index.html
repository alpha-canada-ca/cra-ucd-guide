Menu d'accueil thématique – Système de conception GCWeb

Système de conception GCWeb

Menu d'accueil thématique

Il s'agit d'une configuration pour présenter un ensemble de liens et de descriptions affichés dans des blocs concis. Chaque bloc du menu d'accueil thématique comporte un titre et un bref résumé du contenu de la page de destination. Des blocs de menu d'accueil clairs, bien rédigés, précis et distincts les uns des autres aident les utilisateurs à choisir le bon chemin pour accomplir leurs tâches.

Quand l'utiliser

  • Utilisez les menus d’accueil thématiques comme ensemble de liens vers des sujets ou des contenus de destination.
  • Utilisez ces menus lorsque l'objectif principal de la page est de fournir une façon de naviguer vers des tâches.
  • Les menus d’accueil thématiques sont toujours utilisés dans les modèles suivants :
    • Page d'accueil institutionnelle
    • Page de sujet
    • Page d'index avec navigation de style métro
  • Les menus d’accueil thématiques peuvent être utilisés, mais pas obligatoirement, dans les situations suivantes :
    • Liens vers des sous-sujets
    • Page de campagne
    • Étapes stylisées

Quoi éviter

  • N'utilisez pas de menu d'accueil thématique pour les listes simples qui ne nécessitent pas de description fournissant plus de contexte (par exemple, une liste de provinces et de territoires).
  • N'utilisez pas de menu d'accueil thématique sans respecter les règles spécifiques de longueur et de style.
  • Ne répétez pas les liens qui se trouvent dans le modèle « En demande » (si vous utilisez les deux modèles sur la même page).
  • N'utilisez pas d'icônes ou d'images pour mettre l'accent sur chaque bloc du menu d'accueil thématique.

Contenu et conception

Voici les règles et les options pour créer un menu d'accueil thématique et des directives pour créer un ensemble cohérent de blocs dans ce menu d'accueil thématique.

Structure d'un bloc de menu d'accueil thématique

Un bloc de menu d'accueil thématique comporte le titre du lien au format hyperlien et une description du lien au format texte. Il est présenté de la façon suivante :

Titre du lien : idéalement 45 caractères

Description idéalement de 120 caractères ou moins, liste avec virgules ou phrase, ne terminez pas par un point

Titre du lien

But
  • Fournit une très courte description de la page à laquelle il renvoie.
  • Aide l'utilisateur à décider si cette page l'aidera à accomplir sa tâche.
Longueur
  • Le nombre de caractères optimal pour les liens est 45 (y compris les espaces) dans les deux langues officielles
  • La limite est de 75 caractères (avec les espaces).
  • Conseil : Visez une version anglaise ne dépassant pas 35 caractères (y compris les espaces) pour permettre à la version française de ne pas dépasser 45 caractères (y compris les espaces).
Contenu
  • Le texte du lien doit être logique dans le contexte de la page.
  • Il doit être descriptif, unique dans la page, et clairement distinct des autres liens sur la page.
  • Il n'est pas obligé de correspondre exactement au titre de la page de destination.
  • Il n'est pas obligé d'être unique dans tout le site Canada.ca.
Format
  • Mettez la première lettre du premier mot en majuscule et celle de tous les noms propres également.
  • Ne mettez pas de ponctuation à la fin du titre du lien (par exemple, « Comment présenter une demande de financement » au lieu de « Comment présenter une demande de financement? »).
  • Ajoutez un lien vers la page de destination.
  • Amélioration - Étiquettes : Vous pouvez ajouter des étiquettes au titre du lien pour indiquer le statut du programme ou du service s'il a récemment changé, par exemple « Nouveau » (utilisez l'étiquette d'information) ou « Fermé » (utilisez l'étiquette de danger).
    Consultez : Étiquettes
Options de style
  • Choisissez un style cohérent pour les blocs de menu d'accueil thématique d'une page, tels que les sujets, les produits ou services, les actions ou les groupes d'utilisateurs.
  • Sujets
    • Exemple :
      • Arts et médias
      • Histoire et patrimoine
      • Commerce et placements culturels
      • Sport
  • Produits ou services
    • Exemple :
      • Régime enregistré d'épargne-retraite
      • Comptes d'épargne libre d'impôt
      • Régime enregistré d'épargne-études
      • Régime enregistré d'épargne-invalidité
      • Premier compte d'épargne logement
  • Actions
    • Exemple :
      • Informer l'ARC de la date du décès
      • Demander la prestation de décès du RPC ou de la RRQ
      • Représenter une personne décédée
      • Demander un certificat de décharge
  • Groupes d'utilisateurs
    • Exemple :
      • Particuliers
      • Entreprises
      • Organismes à but non lucratif
      • Non-Canadiens
      • Fiscalistes

Description du lien

But
  • Fournit plus de détails et de précision sur la portée du lien.
  • Aide l'utilisateur à décider si cette page l'aidera à accomplir sa tâche en lui donnant un peu plus d'informations que le titre du lien.
Longueur
  • Les descriptions de lien ne doivent pas dépasser 120 caractères (y compris les espaces) dans les deux langues officielles.
  • Conseil : Visez une version anglaise ne dépassant pas 100 caractères (avec espaces) pour permettre à la version française de ne pas dépasser 120 caractères (avec espaces).
Contenu
  • Décrivez ce que l'utilisateur peut trouver ou faire sur la page à laquelle le titre du lien renvoie.
  • Énumérez les mots clés ou les expressions clés séparés par des virgules plutôt que des phrases complètes (style recommandé).
  • N'utilisez pas de phrases d'introduction comme « Comprend… », « Renseignements sur… » ou « Apprenez-en plus au sujet de… ».
  • Ne répétez pas des renseignements déjà fournis dans le titre du lien.
  • N'incluez pas de messages promotionnels.
Format
  • N'incluez pas de liens.
  • Ne mettez pas de ponctuation à la fin de la description.
  • N'ajoutez pas de formatage supplémentaire aux descriptions (par exemple, gras, puces).
Options de style
  • Liste des liens ou des titres de la page à laquelle on renvoie (style recommandé), séparée par des virgules
    • Conseil : N'incluez pas le mot « et » avant le dernier élément de votre liste.
      • Vous gagnez quelques caractères.
      • Vous n'avez pas à vous demander si une virgule d'Oxford est nécessaire pour que la phrase soit compréhensible.
    • Exemple :
      • Comptes d'épargne libre d'impôt, régimes d'épargne enregistrés, régimes de pension communs, administrateurs de régimes
  • Description des tâches que l'utilisateur peut accomplir
    • Exemple :
      • Produire sa déclaration d'impôt, obtenir une trousse d'impôts, vérifier l'état de son remboursement d'impôt
  • Description de ce que vous pouvez obtenir et des personnes admissibles
    • Exemple :
      • Paiement mensuel pour les familles admissibles ayant des enfants de moins de 18 ans
  • Modification - Menu déroulant :
    • Les menus déroulants sont autorisés à la place d'une description de lien dans ces circonstances :
      • s'ils permettent de fournir un accès à un grand nombre d'éléments au même niveau (par exemple, des listes de pays ou d'espèces)
      • si la page de destination comprend principalement une liste de liens, et que l'on peut entièrement la remplacer par le menu déroulant

Ordre et organisation des blocs de menu d'accueil thématique

  • Fondé sur la demande
    • Priorisez les menus d’accueil thématiques en fonction de la demande (par exemple, en fonction de l'analytique Web), avec des liens à forte demande s'affichant avant ceux à faible demande.
    • Dans une présentation à une seule colonne, la priorité s’établit de haut en bas.
    • Dans une présentation à plusieurs colonnes, la priorité s’établit de gauche à droite, et ensuite de haut en bas.
  • Amélioration : Liste numérotée
    • Utilisez les blocs de menu d'accueil thématique séquentiels et numérotés si les liens doivent être suivis dans un ordre étape par étape (par exemple, confirmer l'admissibilité, s'inscrire, puis imposer la TPS/TVH).
    • Utilisez ce style pour permettre aux personnes de sauter les étapes facultatives ou non applicables ou les étapes déjà effectuées.
    • Utilisez le modèle des étapes stylisées lorsque vous présentez des blocs de menu d'accueil thématique séquentiels sur une page.
    • Consultez : Étapes stylisées
  • Amélioration : Utiliser des sous-titres
    • Utilisez-les pour regrouper les liens et les descriptions qui sont très similaires, et losqu'il ne serait pas logique de déplacer le groupe dans une page de sous-sujets.
    • Utilisez-les pour éviter la surcharge de renseignements; idéalement pas plus de 9 liens avec des descriptions par sous-titre.
    • Consultez Titres et sous-titres pour obtenir des directives sur l'écriture de bons sous-titres.
  • Combinaison des styles précédents, le cas échéant; toutefois, limitez le nombre de fois où vous les combinez pour éviter des ensembles trop complexes de liens et de descriptions.

Combiner les titres et les descriptions des liens

Écrivez des titres et des descriptions de liens qui fonctionnent bien ensemble et qui fonctionnent bien avec les autres blocs de menus d’accueil thématiques de la page.

Choisissez un texte qui permet de distinguer les liens

L'objectif des menus d'accueil thématiques est d'aider les utilisateurs à choisir le meilleur chemin pour accomplir leur tâche.

Rédigez des descriptions de liens qui aident les utilisateurs à distinguer les différents chemins et tâches. Fournissez des informations pertinentes sur ce que les utilisateurs peuvent trouver sous ce lien et sur les personnes admissibles.

Ne rédigez pas des descriptions de liens ayant un texte presque identique car elles n’aideront pas les utilisateurs à déterminer si les liens sont pertinents pour eux. Placez plutôt le texte générique comme phrase d'introduction entre l'en-tête et le menu d'accueil thématique.

Exemples : Menu d'accueil thématique pour les prestations

Rédigez des descriptions de liens qui aident les utilisateurs à distinguer les différents chemins et tâches :

Allocation canadienne pour enfants

Versement mensuel pour les familles admissibles ayant des enfants de moins de 18 ans

Crédit pour la taxe sur les produits et services/taxe de vente harmonisée (TPS/TVH)

Versement trimestriel pour les personnes à revenu faible et modeste, montant et dates de versement

Prestation dentaire canadienne

Prestation provisoire pour les familles admissibles ayant des enfants de moins de 12 ans

Paiement de l'incitatif à agir pour le climat

Versement trimestriel pour les résidents de l'Alberta, de la Saskatchewan, du Manitoba et de l'Ontario

Allocation canadienne pour les travailleurs

Prestation pour les personnes qui travaillent et qui ont un faible revenu, montants et dates de versement

Prestation pour enfants handicapés

Versement mensuel aux familles qui s'occupent d'un enfant de moins de 18 ans ayant une déficience grave

Ne rédigez pas des descriptions de liens ayant un texte presque identique :

Allocation canadienne pour enfants

Qui peut faire une demande, comment faire une demande, le montant que vous pouvez obtenir, les dates de paiement

Crédit pour la taxe sur les produits et services/taxe de vente harmonisée (TPS/TVH)

Qui peut faire une demande, comment faire une demande, le montant que vous pouvez obtenir, les dates de paiement

Prestation dentaire canadienne

Qui peut faire une demande, comment faire une demande, le montant que vous pouvez obtenir, les dates de paiement

Paiement de l'incitatif à agir pour le climat

Qui peut faire une demande, comment faire une demande, le montant que vous pouvez obtenir, les dates de paiement

Allocation canadienne pour les travailleurs

Qui peut faire une demande, comment faire une demande, le montant que vous pouvez obtenir, les dates de paiement

Prestation pour enfants handicapés

Qui peut faire une demande, comment faire une demande, le montant que vous pouvez obtenir, les dates de paiement

Utilisez une structure parallèle

Une structure parallèle signifie que les blocs de menu d'accueil thématique correspondent les uns aux autres sur le plan grammatical et qu'ils partagent d'autres similitudes fondamentales. Par exemple, utilisez soit des verbes ou des noms pour le titre de tous les blocs de menu d'accueil de la page mais pas un mélange des deux.

Exemple : Utilisation d'une structure parallèle pour le texte des liens pour les prestations

Écrivez :

  • Allocation canadienne pour enfants (ACE)
  • Crédit pour la TPS/TVH
  • Prestation dentaire canadienne
  • Paiement de l'incitatif à agir pour le climat
  • Allocation canadienne pour les travailleurs
  • Prestation pour enfants handicapés

N’écrivez pas :

  • Faire une demande d'Allocation canadienne pour enfants (ACE)
  • Paiement trimestriel du crédit pour la TPS/TVH
  • Prestation dentaire canadienne
  • Obtenir le paiement de l'incitatif à agir pour le climat
  • Aperçu de la prestation pour enfants handicapés
