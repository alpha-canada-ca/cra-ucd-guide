Guide de conception axée sur l’utilisateur de l’ARC

Navigation de style métro

L'ARC est le créateur du modèle de navigation de style métro.

Quand l'utiliser

Divisez les tâches longues et complexes en pages qui se concentrent chacune sur une étape ou une réponse spécifique dont les gens ont besoin avant de passer à l'étape ou à la section suivante. Elle est idéale pour présenter des services et des processus.

Regroupez les étapes d'une même tâche et assurez une navigation logique entre les pages. Limitez le nombre d'étapes de la tâche principale à un nombre raisonnable (pas plus de 8).

Version de base du GC

La version de base, qui comprend une page d'index et un niveau d'étapes, fait partie du système de conception Éléments de base du GC.

Pour cette version, utilisez :

  • Code pour la navigation de style métro dans la page d’index principale
  • Code pour la page de contenu de style métro

Cette version est utilisée pour les prestations régulières de l'assurance-emploi.

Variante de l'ARC

L'ARC travaille à améliorer le modèle et la gouvernance, y compris quand et comment utiliser les sous-étapes. Au fur et à mesure que nous faisons évoluer le modèle, des fonctionnalités supplémentaires sont ajoutées et cette documentation sera fournie ici dans les prochains mois.

Voici quelques exemples de variante de l’ARC :

  • Crédit d'impôt pour personnes handicapées (CIPH)
  • Prestation dentaire canadienne
  • Supplément unique à l'Allocation canadienne pour le logement
  • Dépenses de travail à domicile pour les employés
